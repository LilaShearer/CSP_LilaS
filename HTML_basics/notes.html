HTML Advanced Notes

A nested tag is just a tag that exists inside of another tag. Some tags require nesting, other's shouldn't be nested.

An attribute is further instructions for the computer given inside of the opening tag. Such as the ID we gave our div or the charset we use inside of the meta tag.

Lists are just like lists you would make in real life. They require nesting li tags inside of other tags.

The first type of list is a numbered list. It has the 'ol' tag.

  1. This is item 1!
  2. This is item 2!
  3. This is item 3!

The second type of list is an unordered list. It has the 'ul' tag.

  • Cool item
  • Other cool item

There's another type of list. It's called a description list. It has the 'dl' then 'dt' then 'dd' tags.
